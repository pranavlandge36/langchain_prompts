# LangChain Prompt Engineering Experiments

This repository contains small experiments and examples created while learning **prompt engineering concepts using LangChain**.

The goal of this project is to understand how prompts and messages are structured and how conversation history can be handled in LLM applications.

---

## Topics Covered

During this project I explored the following concepts:

* Static Prompts
* Dynamic Prompts
* Prompt Templates
* Messages and Message Types
* Message Placeholders
* Chat History Handling

---

## Concepts Explained

### 1. Static Prompts

Static prompts are fixed prompts where the input does not change.

Example:

```python
prompt = "You are a helpful assistant"
```

These prompts are useful for simple tasks but they lack flexibility.

---

### 2. Dynamic Prompts

Dynamic prompts allow inserting variables inside prompts.

Example:

```python
prompt = f"Explain {topic} in simple terms"
```

Dynamic prompts allow the same template to be reused with different inputs.

---

### 3. Prompt Templates

Prompt templates help structure prompts and make them reusable.

Using LangChain:

```python
from langchain_core.prompts import ChatPromptTemplate

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant"),
    ("human", "{query}")
])
```

Variables such as `{query}` are replaced at runtime.

---

### 4. Messages and Message Types

LangChain uses structured messages for conversations.

Common message types include:

* **System Message** – Defines the behavior of the AI.
* **Human Message** – Input provided by the user.
* **AI Message** – Response generated by the AI.

Example:

```python
HumanMessage(content="Hello")
AIMessage(content="Hi, how can I help?")
```

---

### 5. Message Placeholder

Message placeholders allow inserting **previous chat history** dynamically into prompts.

Example:

```python
MessagesPlaceholder(variable_name="chat_history")
```

This helps maintain context in conversations.

Example conversation structure:

```
System: You are a helpful assistant
Human: Hello
AI: Hi!
Human: What is LangChain?
```
